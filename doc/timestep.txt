:line

timestep command :h3

[Syntax:]

timestep dt :pre

dt = timestep size (seconds) :ul

[Examples:]

timestep 1.0e-5
timestep 0.003 :pre

[Description:]

Set the timestep size for spatial or ode simulations.

[Restrictions:]

You cannot set the timestep for a Gillespie simulation.
