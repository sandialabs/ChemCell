:line

unreact command :h3

[Syntax:]

unreact reaction-ID :pre

reaction-ID = ID of previously defined reaction :ul

[Examples:]

unreact AB
unreact channel-flux :pre

[Description:]

Delete a previously defined reaction so that it is no longer active.

[Restrictions:] none

[Related commands:] none
