species command

Syntax: 

species species-ID alias1 alias2 ...
  • species-ID = user-assigned name for the particle
  • alias1,alias2,etc = optional alias names by which the species can also be referenced

Examples: 

species Ca_cyto
species C8x432 Ca Ca_input Ca_known

Description:

Define a particle species and alternate names by which it can be referred to in any command that takes a species-ID.

The ID of a species and its aliases can contain alphanumeric characters and underscores and dashes.

If species-ID does not exist, it is created with default dimension, diffusivity, and permeability settings. If species-ID already exists, then additional unique aliases must be specified which are associated with that species-ID.

Restrictions: none

Related commands: none

Default: none
