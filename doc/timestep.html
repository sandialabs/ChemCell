timestep command

Syntax: 

timestep dt
  • dt = timestep size (seconds)

Examples: 

timestep 1.0e-5
timestep 0.003

Description:

Set the timestep size for spatial or ode simulations.

Restrictions:

You cannot set the timestep for a Gillespie simulation.

Related commands:

run, run_style

Default: none
