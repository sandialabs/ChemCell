volume command

Syntax: 

volume V
  • V = volume of simulation domain (liters)

Examples:

volume 8.0e-15

Description:

Set the volume of the system for a non-spatial simulation. This will be the well-mixed volume within which reactions take place, e.g. for a stochastic Gillespie or continuum ODE simulation.

Restrictions:

This command can only be used for non-spatial simulations. See the global command for spatial simulations.

Related commands:

global

Default: none
