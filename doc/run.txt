:line

run command :h3

[Syntax:]

run N
run delta :pre

N = run for N timesteps (integer)
delta = run for delta seconds (floating point) :ul

[Examples:]

run 1000
run 5.0 :pre

[Description:]

Run a simulation for N timesteps or delta seconds.  If a previous run
was made, this command continues the simulation.
